# Brutalist Signal — Template Context

## Page Type
Raw precision sales/landing page — bold, direct, information-first design.

## Design System
- **Palette**: Paper (#E8E4DD), Signal Red (#E63B2E), Black (#111)
- **Fonts**: Space Grotesk (headings), DM Serif Display (accents), Space Mono (monospace labels)
- **Style**: Zero border-radius, 2–3px borders, stark contrast, no decoration, GSAP animations
- **Tone**: Direct, confident, unapologetically bold

## Section Structure
1. **Hero** — Paper background + bold black headline + red accent CTA
2. **Problem** — "Sound familiar?" with sharp bullet points
3. **Solution / Features** — 3 flat cards with thick borders
4. **Social Proof** — 2–3 quotes in monospace style
5. **CTA Block** — High-contrast final push
6. **Footer** — Minimal brand line

## Suggested First Edits
- Replace the headline with your core value proposition
- Update the red accent (#E63B2E) to your brand color
- Swap the testimonial quotes with real client feedback
