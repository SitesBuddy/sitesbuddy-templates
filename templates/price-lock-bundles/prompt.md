# Price Lock Bundles — Template Customisation Guide

This is a subscription box / price-stable essentials template. It's built around an anti-inflation narrative — the key message is that customers can lock in today's prices and protect themselves from rising costs.

## Key things to customise

**Brand name**
- Replace "LockedIn" with the actual service name throughout (nav, hero, footer, page title)
- Update the tag emoji and visual identity — the lock emoji (🔒) can be changed

**Pricing**
- Update the 3 plan prices: Starter (€39), Family (€69), Community (€119)
- Edit plan names, taglines, and feature lists to match what's actually offered
- The "most popular" badge on the Family plan can be moved or removed

**Urgency stats**
- The urgency banner and "why now" cards include statistics (grocery prices up 18%, fuel up 22%, etc.)
- Keep or update these to match current real data — or generalise them if you prefer a softer tone

**What's in the bundle**
- The contents grid shows 12 items from the "Family bundle" — update to reflect what's actually included
- Each item has an emoji, name, and quantity per month

**Delivery area**
- Update the delivery radius (currently "within 25km of Millbrook") in the guarantee row and FAQ

**FAQ**
- Update the 4 FAQ answers to reflect the actual cancellation policy, swap rules, and delivery details

**Contact / business info**
- Update the footer email (hello@lockedin.ie) and any other contact details
- Update the company registration name if shown

**The "43 people locked in" counter**
- This is a static number — update it to something realistic, or remove it
- A developer can make this pull from a real database if needed

**Colours**
- `--navy: #0d1f38` — dark navy background
- `--green: #00c853` — electric green (primary action colour)
- Dark theme throughout — suitable for an anti-inflation / deal-focused brand

## What this template works for
- Subscription grocery box delivery
- Bulk essentials pre-order programme
- Community buying club with fixed memberships
- Any service promising price stability on regular household purchases
