# Primary Classroom HQ

Build a welcoming, parent-facing primary classroom website for one teacher and class.

The page should help families find current learning, upcoming dates, routines, optional home practice, and the correct contact route without duplicating a private learning-management system. Keep private student information out of the public page.

Required sections:

- Fixed navigation with one family-form CTA
- Clear weekly learning hero
- Marquee summarizing current reminders
- This-week learning snapshot
- Readable class calendar
- Parent routine and policy information
- Short current resource library
- Privacy-aware contact section

Use warm editorial classroom photography, an optimistic primary-school palette, clear mobile behavior, SitesBuddy structural hooks, accessible navigation, and editable links.
