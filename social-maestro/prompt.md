# Social Maestro — Template Context

## Page Type
Bold editorial portfolio page — social media manager / creative professional.

## Design System
- **Palette**: Red (#E85D4E), Charcoal (#2A2A2A), Off-white (#FDFBF7)
- **Fonts**: Strong sans-serif for headings, clean body type
- **Style**: Portfolio-focused layout, strong typography hierarchy, editorial grid, GSAP animations
- **Tone**: Confident, professional, results-driven

## Section Structure
1. **Hero** — Bold headline + professional tagline + CTA to view work
2. **Problem** — What businesses struggle with (social media pain points)
3. **Portfolio / Features** — Service cards or project showcases
4. **Social Proof** — Client testimonials with results
5. **CTA Block** — Hire me / book a call
6. **Footer** — Contact info + social links

## Suggested First Edits
- Replace the headline with your name and professional title
- Add your real client testimonials and results
- Swap the red accent (#E85D4E) to your personal brand color
