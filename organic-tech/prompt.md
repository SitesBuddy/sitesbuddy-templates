# Organic Tech — Template Context

## Page Type
Clinical boutique sales/landing page — lab meets luxury magazine.

## Design System
- **Palette**: Moss (#2E4036), Clay (#CC5833), Cream (#F2F0E9)
- **Fonts**: Plus Jakarta Sans (headings), Cormorant Garamond (serif accents), IBM Plex Mono (labels)
- **Style**: Light background, earthy organic tones, clay accent pops, GSAP scroll animations
- **Tone**: Trusted, natural, scientific yet approachable

## Section Structure
1. **Hero** — Cream backdrop + moss green headline + clay accent CTA
2. **Problem** — 3 pain points the target audience recognizes
3. **Solution / Features** — 3 feature cards with inline SVG icons
4. **Social Proof** — 2–3 testimonials
5. **CTA Block** — Final offer push
6. **Footer** — Brand + tagline

## Suggested First Edits
- Swap the headline to describe your product or service
- Change the clay accent color to match your brand
- Update the feature card icons and descriptions
